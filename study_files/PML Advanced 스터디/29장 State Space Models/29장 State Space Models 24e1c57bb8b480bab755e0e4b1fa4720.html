29.6 Linear Dynamical Systems (LDS)

이 장에서는 구체적으로 Linear-Gaussian State-Space Model (LG-SSM), a.k.a LDS 에 대해서 다룹니다.

LG-SSM 이기 위해서는,

  • Transition Function f\boldsymbol ff﻿ + Observation Function h\boldsymbol hh﻿ 들이 Linear 할 때.
  • Process Noise qt\boldsymbol q_tqt​﻿ + Observation Noise rt\boldsymbol r_trt​﻿ 들이 Gaussian 일 때.

이며 그래프 모델 자체는 우측과 같습니다.

정의 자체는 그래서 다음과 같이 해줄 수가 있습니다.

Bias 가 없고 Input 이 없다면 모델은 다음과 같이 Simplify 됩니다.

그러면 이것을 방정식으로 다음과 같이 적어줄 수 있게 됩니다.

  • qt∼N(0,Qt):\boldsymbol{q}_t \sim \mathcal{N}\left(\mathbf{0}, \mathbf{Q}_t\right) :qt​∼N(0,Qt​):﻿ Process Noise
  • rt∼N(0,Rt):\boldsymbol{r}_t \sim \mathcal{N}\left(\mathbf{0}, \mathbf{R}_t\right) :rt​∼N(0,Rt​):﻿ Observation Noise

여기에서 저희의 파라미터들은 그러면 다음과 같습니다 (보통은 ttt﻿ 에 의존하지 않는다는 가정이긴 함).

29.7 LDS: Applications

29.7.1 Object Tracking

2차원에서 놀고 있는 Object 을 생각해보겠습니다. 이때 Position 이 (ut,vt)(u_t, v_t)(ut​,vt​)﻿ 그리고 Velocity 를 (u˙t,v˙t)\left(\dot{u}_t,\dot{v}_t\right)(u˙t​,v˙t​)﻿ 으로 두면 State at time ttt﻿ 를 zt=(utvtu˙tv˙t)\boldsymbol{z}_t=\left(\begin{array}{llll}u_t & v_t & \dot{u}_t & \dot{v}_t\end{array}\right)zt​=(ut​​vt​​u˙t​​v˙t​​)﻿ 으로 둘 수가 있습니다.

Discrete Time 만 고려하면 Discrete Time SSM 은 다음과 같습니다.

  • Δ:\Delta :Δ:﻿ Step Size between Consecutive Discrete Measurement Times
  • qt∼N(0,Q):\boldsymbol{q}_t \sim \mathcal{N}(\mathbf{0}, \mathbf{Q}) :qt​∼N(0,Q):﻿ Process Noise
  • Noise Covariance Matrix Q:\mathbf Q :Q:﻿

해석 :

  • Position 부분에만 Δ\DeltaΔ﻿ 가 Velocity 와 함께 직접적으로 더해지며 이는 Velocity 가 Position 에 관한 함수가 아니기에.
  • qt\boldsymbol q_tqt​﻿ 부분에 Δ\DeltaΔ﻿ 가 들어가며, 이는 Random Acceleration 이 Position + Velocity 에 어떤 영향을 끼치는지 관련이다.
    • 우하단 : Velocity 는 Acceleration 한 번 적분한 것이기에 Velocity Variance 는 시간에 따라서 Linear 하다.
    • 좌상단 : Position 은 Acceleration 두 번 적분한 것이기에 Position Variance 는 시간에 따라서 Cubic 하다.
    • Off-Diagonals : Position 과 Velocity 의 Covariance 부분, 시간에 따라서 Quadratic 하다.

이러한 세팅에서 Location 을 관측한다고 해보자. 이때 Location 에 대한 Gaussian Noise 까지 있다고 하면, 다음과 같이 Observation Model 을 구축할 수가 있다.

  • rt∼N(0,R):\boldsymbol{r}_t \sim \mathcal{N}(\mathbf{0}, \mathbf{R}) :rt​∼N(0,R):﻿ Observation Noise

이것의 움직임을 (a), Kalman Filter 을 통해서 p(zt∣y1:t,θ)p(\boldsymbol z_t \mid \boldsymbol y_{1:t}, \boldsymbol \theta)p(zt​∣y1:t​,θ)﻿ 을 구하면 (b) 와 같이, 그리고 Kalman Smoother 까지 활용해서 구하면 (c) 와 같이 나옵니다.
