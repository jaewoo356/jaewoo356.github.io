Using Inverse & Rejection Methods

Multivariate Inverse Transform Method

어떠한 다변량 분포에 대한 CDF 를 F1,⋯ ,d(x1,⋯ ,xd)F_{1, \cdots, d} (x_1, \cdots , x_d)F1,⋯,d​(x1​,⋯,xd​) 가 있다면 이것을 다음과 같이 쪼개는 것이 가능합니다.

그러면 저희는 다음과 같이 Inverse Transform Method 를 적용하는 것이 가능합니다.

물론 이렇게 진행하면 각각에 대한 식을 구해야 된다는 단점이 존재하며, ddd 크기가 커지면 커질수록 이 식을 구하는 것은 Intractable 해집니다.

mu    <- c(0, 0, 0)
Sigma <- matrix(c(
  1.0, 0.4, 0.2,
  0.4, 1.5, 0.3,
  0.2, 0.3, 2.0
), nrow = 3, byrow = TRUE)

sample_mvnorm_inverse <- function(mu, Sigma) {
  # X1 ~ N(mu1, Sigma[1,1])
  u1 <- runif(1); z1 <- qnorm(u1)
  x1 <- mu[1] + sqrt(Sigma[1,1]) * z1
  
  # X2 | X1=x1 ~ N(m21, v21)
  m21 <- mu[2] + (Sigma[2,1]/Sigma[1,1]) * (x1 - mu[1])
  v21 <- Sigma[2,2] - Sigma[2,1]^2 / Sigma[1,1]
  u2  <- runif(1);   z2 <- qnorm(u2)
  x2  <- m21 + sqrt(v21) * z2
  
  # X3 | (X1,X2) ~ N(m312, v312)
  S11  <- Sigma[1:2, 1:2]
  S21  <- Sigma[3, 1:2]
  S12  <- Sigma[1:2, 3]
  invS <- solve(S11)
  delta <- c(x1, x2) - mu[1:2]
  m312  <- mu[3] + as.numeric(S21 %*% invS %*% delta)
  v312  <- Sigma[3,3] - as.numeric(S21 %*% invS %*% S12)
  u3    <- runif(1); z3 <- qnorm(u3)
  x3    <- m312 + sqrt(v312) * z3
  
  c(x1, x2, x3)
}

손으로 연산만 해도 굉장히 힘들며 오래 걸립니다. 식을 매번 구해야 된다는 단점이 있기 때문이죠. 추가적으로 Inverse CDF 도 매번 구해야 되는 단점도 있으며 이게 존재하지 않을 수도 있습니다.

반면에 Rejection Method 를 다변량에 적용시키면 어떨까요? 똑같은 방법으로 적용시키면 되지만, 좋은 Proposal Density qqq 를 구하는 것이 어려우며, 대부분의 데이터가 기각당하는 경우도 빈번합니다. 그래서 이를 다르게 활용한 방법으로 MCMC 가 존재하며, 이는 나중에 다루도록 하겠습니다.

Multivariate Normal Distribution

다변량 정규 분포 X∼Nn(μ,Σ)\mathbf{X} \sim \mathrm{N}_n\left(\boldsymbol{\mu}, \mathbf{\Sigma}\right)X∼Nn​(μ,Σ) 는 다음과 같은 pdf 를 가집니다.

하지만 정규분포들끼리는 서로 선형적인 관계를 가진다는 것을 저희는 알고 있습니다. 다음과 같은 독립적인 분포의 다변량 표준정규분포는 쉽게 뽑을 수가 있습니다. 그저 표준정규분포에서 i.i.d 하게 뽑은 다음에 그것을 벡터 형태로 바꿔주면 되죠.

X\mathbf{X}X 에 대해서는 그러면 선형적인 관계임을 이용해서 다음과 같이 구할 수가 있습니다.

이때의 가정은 AA⊤\mathbf{AA}^\topAA⊤ 가 Positive Definite 이라는 가정이 들어갑니다.

그러면 만약에 X\mathbf{X}X 에서 뽑고 싶으면 그저 Σ=AA⊤\mathbf{\Sigma} = \mathbf{AA}^\topΣ=AA⊤ 인 Matrix A\mathbf{A}A 를 찾은 다음에 X:=AZ+μ\mathbf{X} := \mathbf{AZ}+\boldsymbol \muX:=AZ+μ 으로 세팅해준다면 X\mathbf{X}X 을 뽑을 수가 있게 됩니다.

이때 Cholesky Decomposition 을 활용해서 A\mathbf{A}A 를 찾으면 더 효율적이게 됩니다.

Cholesky Decomposition 이란 Σ\mathbf{\Sigma}Σ 와 같은 Symmetric + Positive Definite Matrix 를 Decompose 하기 위한 굉장히 효율적인 방법이며, 이렇게 하게 되면 A\mathbf{A}A 가 Unique 한 하삼각행렬로 Σ=AA⊤\mathbf{\Sigma} = \mathbf{AA}^\topΣ=AA⊤ 으로 구성하는 것이 가능해집니다.

이것을 위의 예시 문제에 다시 한번 적용시켜보겠습니다.

n <- 10000
d <- 3

z_vals <- qnorm(runif(n * d))
z_mat <- matrix(data = z_vals, nrow = 3)

mu    <- c(0, 0, 0)
Sigma <- matrix(c(
  1.0, 0.4, 0.2,
  0.4, 1.5, 0.3,
  0.2, 0.3, 2.0
), nrow = 3, byrow = TRUE)

A <- t(chol(Sigma))

X <- A %*% z_mat + mu

이렇게 진행하게 된다면 위의 Inverse CDF 방법보다 더 빠르게 샘플링을 할 수 있게 됩니다.

Copulas (Joint Distributions)

Copula 란 그저 [0,1]d[0,1]^d[0,1]d 위에 있는 Joint Distribution 을 의미합니다. 쉽게 말하자면, CDF 의 다변량 버전이며, correlation 까지 알려주는 cdf 함수라고 생각하면 됩니다.

예시로 2-차원일 때의 Copula 는 C(x,y),  0≤x≤1,  0≤y≤1C(x,y), \; 0 \le x \le 1, \; 0\le y \le 1C(x,y),0≤x≤1,0≤y≤1 이며, 다음과 같은 특성들을 가집니다.

  • C(x,0)=0,C(0,y)=0,C(x,0) = 0, \quad C(0, y) = 0,C(x,0)=0,C(0,y)=0,
  • C(x,1)=x,C(1,y)=yC(x,1) = x, \quad C(1,y) = yC(x,1)=x,C(1,y)=y

이 예시를 이어가서 만약에 Joint Distribution 이 (X,Y)∼H(X,Y) \sim H(X,Y)∼H 이라고 하며 Marginal Distribution 들이 X∼FX\sim FX∼F, Y∼GY\sim GY∼G 이라고 한다면, HHH 의 Copula 인 CHC_HCH​ 는 다음과 같이 표현이 가능합니다.

이를 Sklar’s 정리에 의해서 Copula 도 알고 Marginal 도 안다면 다음과 같이 Joint Distribution 인 H(x,y)H(x,y)H(x,y) 도 다음과 같이 구할 수가 있습니다.

이게 무슨 말이냐면, (U1,U2)(U_1, U_2)(U1​,U2​) 를 CHC_HCH​ 에서 뽑을 수가 있다면, 그러면 U1U_1U1​ 과 U2U_2U2​ 는 상관관계가 있는 확률 값들이며, 이들을 Inverse Transform Method 으로 X=F−1(U1),  Y=G−1(U2)X = F^{-1}(U_1), \; Y = G^{-1}(U_2)X=F−1(U1​),Y=G−1(U2​) 으로 세팅하게 된다면 Joint Distribution 에서 뽑을 수 있게 됩니다.

반대로 얘기하자면, (X,Y)∼H(X,Y) \sim H(X,Y)∼H 에서 뽑을 수 있게 되면, U1=F(X),  U2=G(Y)U_1 = F(X), \; U_2 = G(Y)U1​=F(X),U2​=G(Y) 으로 세팅하게 된다면 (U1,U2)∼CH(U_1, U_2) \sim C_H(U1​,U2​)∼CH​ 에서 뽑게 되는 셈이랑 똑같게 된다는 것입니다.

예시로 Gaussian Copula 를 보겠습니다.

Gaussian Copula

Copula 는 그저 [0,1]d[0,1]^d[0,1]d 위에 있는 Joint Distribution 를 의미한다고 했습니다. 그러면 Gaussian Copula 란, 다음과 같은 (X,Y)(X,Y)(X,Y) 가 있다고 하겠습니다.

그러면 Φ(X)\Phi(X)Φ(X) 와 Φ(Y)\Phi(Y)Φ(Y) 의 Joint Distribution 을 Gaussian Copula 이라고 부릅니다. 예시로 ρ=0.5\rho = 0.5ρ=0.5 일 때의 Gaussian Copula 를 뽑겠습니다.

rm(list=ls())
n <- 20000
rho = 0.5

x <- qnorm(runif(n))
z_2 <- qnorm(runif(n))
y <- x * rho + sqrt(1 - 
      rho^2) * z_2

u_1 <- pnorm(x)
u_2 <- pnorm(y)

plot(u_1, u_2, cex = 0.25, 
		pch = 20)

이런 방식으로 Copula 에서 뽑는 것이 가능합니다. 위의 코드에서는 사실 다음과 같이 흘러갑니다.

  • 먼저 XXX 를 뽑아준다.
  • 그 다음에 YYY 를 뽑는데, 이때 Y=ρσ2Z1+σ21−ρ2Z2+μ2Y = \rho \sigma_2 Z_1 + \sigma_2 \sqrt{1-\rho^2}Z_2 + \mu_2Y=ρσ2​Z1​+σ2​1−ρ2​Z2​+μ2​ 인 점을 활용합니다.
  • U1:=F(X)U_1 := F(X)U1​:=F(X) 으로 세팅합니다.
  • U2:=G(Y)U_2 := G(Y)U2​:=G(Y) 으로 세팅합니다.

위의 방법이 사실상 Copula 부분에서 설명한 것이랑 정확히 똑같습니다.

이 방법을 활용시켜서 다음과 같은 예시에 적용해보겠습니다.
