7장 Sufficiency (1)

Properties of Point Estimators

Consistency : Yn→PθY_n \xrightarrow P \thetaYn​P​θ 이 만족이 된다면 Consistent Estimator 이라고 불린다!

Unbiasedness : E⁡[Yn]=θ\operatorname E \left[Y_n \right] = \thetaE[Yn​]=θ 이 만족이 된다면 Unbiased Estimator 이라고 불린다!

여기에서 YYY 은 뭘까요? 바로 저희들이 세운 표본에 의한 추정량입니다.

모든 MLE 들이 Unbiased 한 것은 아니지만 대부분의 MLE 들은 Asymptotically Unbiased 하다!

예:

Minimum Variance Unbiased Estimator (MVUE)

저희는 추정량을 Y=u(X1,X2,⋯ ,Xn)Y = u(X_1, X_2, \cdots, X_n)Y=u(X1​,X2​,⋯,Xn​) 으로 세울 수가 있습니다. 이 추정량이 만약에 우도함수를 최대화 시켜주는 추정량이면 그것을 Maximum-Likelihood Estimator 이라고 불렀습니다. 그러면 MVUE 는 무엇일까요?

MVUE (Minimum Variance Unbiased Estimator) 의 경우 비슷하게 Y=u(X1,X2,⋯ ,Xn)Y = u(X_1, X_2, \cdots, X_n)Y=u(X1​,X2​,⋯,Xn​) 이 어떠한 모수 θ\thetaθ 의 MVUE 이기 위해서는 다음과 같은 조건이 만족되어야 합니다.

  • Unbiased : E⁡[Y]=θ\operatorname E[Y] = \thetaE[Y]=θ
  • Minimum Variance : Var⁡(Y)≤Var⁡(Y2),\operatorname{Var}(Y) \le \operatorname{Var}(Y_2), Var(Y)≤Var(Y2​), where Y2Y_2Y2​ is ANY other Estimator of θ\thetaθ.

예시 :

위의 예시에 의하면 Xˉ\bar XXˉ 와 X1X_1X1​ 둘다 Unbiased 한 추정량입니다. 하지만 어떤게 더 낮은 분산을 가지고 있을까요? 여기에서는 당연히 Xˉ\bar XXˉ 입니다.

그렇다면 Xˉ\bar XXˉ 가 MVUE 라고 단정지을 수가 있나요? 없습니다! 모든 추정량들과 함께 비교를 해야되기 때문입니다.

Decision Function, Loss Function, Risk Function

Y=u(X1,⋯ ,Xn)Y = u(X_1, \cdots, X_n)Y=u(X1​,⋯,Xn​) 이 θ\thetaθ 에 대한 점추정을 하기 위해서 필요한 통계량이라고 하겠습니다.

δ(y)\delta(y)δ(y) 는 그러면 관측된 YYY 에 대한 함수라고 두겠습니다. δ(y)\delta(y)δ(y) 는 그러면 θ\thetaθ 에 대한 점추정을 결정해주는 함수라고 봐도 됩니다.

이러한 함수 δ\deltaδ 를 저희는 Decision Function / Decision Rule 이라고 부르며, Decision Function 의 1 값을 1개의 Decision 이라고 부릅니다.

Loss Function 이란 모수 θ\thetaθ 와 Decision δ(y)\delta(y)δ(y) 의 차이를 의미하며, 사용할 수 있는 Loss 함수는 굉장히 다양합니다. 머신러닝에서 아시는 Loss 함수를 적용시켜도 되기는 합니다!

식으로는 L[θ,δ(y)]L\left[\theta, \delta(y)\right]L[θ,δ(y)] 으로 두며 이것의 역할은 Decision 이 얼마나 정확한지를 나타내주는 지표입니다.

Risk Function 은 이 Loss Function 의 기댓값입니다.

여기에서 Loss Function 이랑의 차이점이 뭐가 있냐면 Loss Function 의 경우에는 Observed Value yyy 가 δ\deltaδ 에 들어갑니다. 반면에 Risk Function 의 경우 모수 θ\thetaθ 와 Decision Function δ,δ(Y)\delta, \delta(Y)δ,δ(Y) 만 들어가게 됩니다. 이것은 yyy 에 대한 기댓값이기에 이렇게 반영이 됩니다.

그럼 이것을 활용시켜서 무엇을 할 수 있을까요?

Solution 1 :

Restriction - E⁡[δ(Y)]=θ,for all θ∈Ω\operatorname E\left[\delta(Y) \right] = \theta, \quad \text{for all } \theta \in \OmegaE[δ(Y)]=θ,for all θ∈Ω

Loss (Example) - L[θ,δ(y)]=[θ−δ(y)]2L\left[\theta, \delta(y) \right] = \left[\theta - \delta(y) \right]^2L[θ,δ(y)]=[θ−δ(y)]2

Risk Function - R[θ,δ]=E⁡[(θ−δ(Y))2]=Var⁡[δ(Y)]R\left[\theta, \delta \right] = \operatorname E\left[\left(\theta - \delta(Y) \right)^2 \right] = \operatorname{Var}\left[\delta (Y)\right]R[θ,δ]=E[(θ−δ(Y))2]=Var[δ(Y)]

여기에서 Risk Function 을 최소화 시키는 δ\deltaδ 가 MVUE 인 δ\deltaδ 값을 찾는 것과 동일합니다!

Solution 2 :

Minimax Principle, R(θ,δ(Y))R(\theta, \delta(Y))R(θ,δ(Y)) 의 최댓값을 최소화 시켜주는 δ\deltaδ 함수가 BEST δ\deltaδ 함수가 됩니다!

예시

Let R(θ,δ1)=125R(\theta, \delta_1) = \frac 1 {25}R(θ,δ1​)=251​ and R(θ,δ2)=θ2R(\theta, \delta_2) = \theta^2R(θ,δ2​)=θ2

Minimax Principle 에 의하면 δ1\delta_1δ1​ 이 δ2\delta_2δ2​ 에 비해서 더 좋다!

Sufficiency

위의 가정을 두면서 통계량 Y=u(X1,X2,⋯ ,Xn)Y = u(X_1, X_2, \cdots, X_n)Y=u(X1​,X2​,⋯,Xn​) 에 대한 얘기를 하겠습니다.

통계량 YYY 가 Sufficient 한 Statistic 이 되기 위해서는 X1,⋯ ,XnX_1, \cdots, X_nX1​,⋯,Xn​ 의 Conditional Distribution by YYY 가 θ\thetaθ 에 의존하면 안됩니다!

교재 정의 :

에서 H(x1,x2,⋯ ,xn)H(x_1, x_2, \cdots, x_n)H(x1​,x2​,⋯,xn​) 이 θ\thetaθ 에 의존하지 않는 것!

예시를 몇개를 들겠습니다.

  • X1,⋯ ,Xn∼iidB⁡(1,θ)→Y1=∑i=1nXiX_1, \cdots, X_n \stackrel{iid}{\sim} \operatorname B(1, \theta) \rightarrow Y_1 = \sum_{i=1}^{n} X_iX1​,⋯,Xn​∼iidB(1,θ)→Y1​=∑i=1n​Xi​
  • X1,⋯ ,Xn∼iidGamma(2,θ)→Y1=∑i=1nXiX_1, \cdots, X_n \stackrel{iid}{\sim} \text{Gamma}(2, \theta) \rightarrow Y_1 = \sum_{i=1}^{n} X_iX1​,⋯,Xn​∼iidGamma(2,θ)→Y1​=∑i=1n​Xi​

근데 사실 위의 방식으로 Sufficient Statistic 을 구하는 것은 매우 어려운 일입니다. 그렇기 때문에 Factorization Theorem 이라는 정리를 활용하면 굉장히 손쉽게 Sufficient Statistic 을 구할 수 있습니다.

Factorization Theorem

X1,⋯ ,Xn∼iidf(x;θ),θ∈ΩX_1, \cdots, X_n \stackrel{iid}{\sim} f(x; \theta), \quad \theta \in \OmegaX1​,⋯,Xn​∼iidf(x;θ),θ∈Ω

위와 같은 상황이 주어진다면 우도 함수를 구축하는 것이 가능합니다.

여기에서 우도 함수를 θ\thetaθ 와 연관된 부분과 θ\thetaθ 와 연관이 없는 부분들로 나눌 수가 있습니다. 여기에서 k1k_1k1​ 과 k2k_2k2​ 가 Non-negative 하다면 u(X1,⋯ ,Xn)u(X_1, \cdots, X_n)u(X1​,⋯,Xn​) 은 Sufficient 한 Statistic 이라고 말할 수가 있습니다.

이것은 if and only if 이기 때문에 모든 Sufficient Statistic 은 이것을 만족시켜줍니다.

예: (정규분포)

그러면 여기에서 저희는 u(X1,⋯ ,Xn)u(X_1, \cdots, X_n)u(X1​,⋯,Xn​) 를 다음과 같이 구성할 수 있습니다.

저희가 흔히 알고 있는 이것에 대한 MLE 는 Xˉ\bar XXˉ 입니다. 하지만 Xˉ\bar XXˉ 도 위의 통계량으로 구축이 가능합니다.

그럼 여기에서 알 수 있는 것을 무엇일까요? 바로 Sufficient Statistic 은 Unique 하지 않다는 것입니다! 따지고 보면 Xˉ\bar XXˉ 도 Sufficient Statistic 이기 때문에 Xˉ2,Xˉ3,log⁡(Xˉ)\bar X ^2, \bar X ^3, \log(\bar X)Xˉ2,Xˉ3,log(Xˉ) 등도 모두 다 Sufficient 한 Statistic 이라는 것입니다.

근데 저희는 MVUE 에 대해서 궁금해했는데, 왜 갑자기 Sufficient Statistic 에 대해서 배울까요? MVUE 를 구하기 위한 빌드업입니다.

Rao-Blackwell Theorem

Y1=u1(X1,⋯ ,Xn)Y_1 = u_1(X_1, \cdots, X_n)Y1​=u1​(X1​,⋯,Xn​) : Sufficient Statistic for θ\thetaθ

Y2=u2(X1,⋯ ,Xn)Y_2 = u_2(X_1, \cdots, X_n)Y2​=u2​(X1​,⋯,Xn​) : Unbiased Estimator of θ\thetaθ

위와 같은 상황을 고려해주시길 바랍니다. 그러면

ψ(Y1)=E⁡[Y2∣Y1]\psi(Y_1) = \operatorname E\left[Y_2 | Y_1 \right]ψ(Y1​)=E[Y2​∣Y1​] 는 θ\thetaθ 의 Unbiased Estimator 가 됩니다. 왜 일까요? 이것은 Conditional Expectation 의 정의를 생각하면 됩니다.

그럼 E⁡[Y2∣Y1]\operatorname E[Y_2|Y_1]E[Y2​∣Y1​] 는 정확하게 뭘까요? 다른게 아니라 그냥 Y1Y_1Y1​ 에 대한 함수입니다.

그렇기 때문에 ψ(Y1)\psi(Y_1)ψ(Y1​) 으로도 표현할 수 있다는 것입니다.

추가적으로 다음과 같은 관계도 성립이 됩니다. (Law of Total Variance)

위의 E⁡(Y2∣Y1)\operatorname E(Y_2|Y_1)E(Y2​∣Y1​) 에 ψ(Y1)\psi(Y_1)ψ(Y1​) 을 대입시키게 된다면 다음과 같이 나오게 됩니다.

Equality 는 Y2Y_2Y2​ 가 Y1Y_1Y1​ 에 대한 함수일 때 해당이 됩니다.

이것을 통해서 저희는 MVUE 를 구하는데 조금 더 편하게 구할 수가 있게 됩니다.

Law of Total Variance :

위의 정리를 한번 더 정리를 하자면,

Y1=u1(X1,⋯ ,Xn)Y_1 = u_1(X_1, \cdots, X_n)Y1​=u1​(X1​,⋯,Xn​) : Sufficient Statistic for θ\thetaθ

Y2=u2(X1,⋯ ,Xn)Y_2 = u_2(X_1, \cdots, X_n)Y2​=u2​(X1​,⋯,Xn​) : Unbiased Estimator of θ\thetaθ

위와 같은 상황에서, ψ(Y1)=E⁡[Y2∣Y1]\psi(Y_1) = \operatorname E\left[Y_2 | Y_1 \right]ψ(Y1​)=E[Y2​∣Y1​] 는 θ\thetaθ 에 대한 Unbiased Estimator 이면서 Sufficient 한 Statistic, 그리고 분산이 다음과 같은 관계를 가집니다.

이렇게 한다면 최소의 분산을 구축하는데에 도움이 훨씬 많이 됩니다.

그럼 왜 ψ(Y1)\psi(Y_1)ψ(Y1​) 는 θ\thetaθ 에 대해서 독립일까요?

적분 식에서 좌측과 우측 둘다 θ\thetaθ 에 대해서 독립적입니다. 그렇기 때문에 전체 식도 θ\thetaθ 에 대해서는 독립적이라는 뜻입니다.
